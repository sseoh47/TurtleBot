# TurtleBot
Arduino-based control system for TurtleBot3 Burger

## System Architecture

### Hardware Components
1. Raspberry Pi 5
2. Arduino Uno
3. Dynamixel Shield
4. Dynamixel XL430-W250-T

### Software Components

1. AI Perception
   - Object detection
   - Lane detection

2. Control System
   - Driving decision logic

3. Communication
   - UART (Software Serial) protocol between Raspberry Pi and Arduino

4. Motor Control
   - Arduino firmware controlling Dynamixel motors
